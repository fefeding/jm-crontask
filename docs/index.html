部署

目录部署，只需修改一次devops.config.js，后面就不用再管了，本地调试用的是根目录，发布后就采用了目录部署。

// devops.config.js
/**
 * 发布部署相关配置
 */
module.exports = {
    /**
     * 部署目录
     * 例如： demo
     * 只有在正式环境才需要
     */
    prefix: process.env.NODE_ENV== 'development'? '' : 'demo'
}
